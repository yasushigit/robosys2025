## 説明
標準入力から数字を読み込み以下の要素を出力する

1行目:数列の合計を足し合わせにより求める

2行目:数列の平方の合計を足し合わせにより求める

3行目:数列の合計をシグマ計算( $\frac{1}{2}n(n+1)$ )により求める

4行目:数列の平方の合計をシグマ計算( $\frac{1}{6}n(n+1)(2n+1)$ )により求める

## 必要なソフトウェア
- Python
  - テスト済み: 3.7〜3.10

- Ubuntu 24.04
